Links help you travel across the web!

Links in HTML are denoted by the <a> </a> tag set. Known as anchor tags. Here is a list of some common attributes used to modify anchor tags...

  • href=URL or #ID
    • The href attribute descibes the links destination. When you click on a link , this is where you will go.
    • Here is an example...
      • <a href=https://www.google.com/> Google</a>

        Would evaluate to the next item on this list with the help of CSS.
      • Google  <-- This is the link made from the anchor tag above.

  • target="_blank" or "_parent" or "_self" or "_top"
    • The target attribute describes how the link opens. When target is set to "_blank" it opens the link in a new tab. That would look like this...
      • <a href=https://www.google.com/ target="_blank"> Google</a>
      • The above tag would evaluate to a link like this -> Google

        If you click the link above it will open iin a new tab.

Lists are very useful!

There are 3 different types of lists in HTML...

  1. Unordered Lists
    • Unordered lists use the tags <ul></ul> to define themselves and <li></li> to define the items on the list. (know as list items) These lists use bullets or other symbols to denote the beginning of a new list item.
    • Here is an example of an unordered list...
      • Item One
      • Item Two
      • Item Three
    • The HTML looks like this...
      • <ul>
           <li>Item One</li>
           <li>Item Two</li>
           <li>Item Three</li>
        </ul>
  3. Ordered Lists
    • Ordered lists use the tags <ol></ol> to define themselves and <li></li> to define the items on the list. (know as list items)These lists use numbers by default to signify the beginning of a new list item.
    • Here is an example of an ordered list...
      1. Item One
      2. Item Two
      3. Item Three
    • The HTML looks like this...
      • <ol>
           <li>Item One</li>
           <li>Item Two</li>
           <li>Item Three </li>
        </ol>
  5. Description Lists
    • Description lists use the tags <dl></dl> to define themselves and <dt></dt> to define the terms of the list as well as <dd></dd> to define the definitions.
    • Here is an example of a description list...
    • Item One
      Description One
      Item Two
      Description Two
      Item Three
      Description Three
    • The HTML looks like this...
      • <dl>
           <dt>Item One</dt>
           <dd>Description One</dd>
           <dt>Item Two</dt>
           <dd>Description Two</dd>
           <dt>Item Three </dt>
           <dd>Description Three</dd>
        </dl>

You can also change the styles of lists with CSS!

  • Different unordered lists styles are achieved like this in the CSS file...

    ul{
      list-style-type:square;
    }

    (circle, disc and none are also alternate styles in addition to square.)
  • Here is a unordered list with the square list-style-type...
    • These
    • Bullets
    • Are
    • Sqaure
  • Here is one with circles...
    • These
    • Bullets
    • Are
    • Sqaure

Get Organized with Tables!

Tables have a <table> tag, a <th> tag (table head), a <tr> tag (table row) and a <td> tag (table data).


Here is an example table...


ID# Name Password Email
12345 Jon pqwe4! jon@email.com
56789 June asdfas june@email.com
14356 Jill fffeww jill@email.com

The HTML code looks like this...

                        
                    <table>
                        <tr id=rowOne>
                            <th>ID#</th>
                            <th>Name</th>
                            <th>Password</th>
                            <th>Email</th>
                        </tr>
                        <tr id=rowTwo>
                            <td>12345</td>
                            <td>Jon</td>
                            <td>pqwe4!</td>
                            <td>jon@email.com</td>
                        </tr>
                        <tr id=rowThree>
                            <td>56789</td>
                            <td>June</td>
                            <td>asdfas</td>
                            <td>june@email.com</td>
                        </tr>
                        <tr id=rowFour>
                            <td>14356</td>
                            <td>Jill</td>
                            <td>fffeww</td>
                            <td>jill@email.com</td>
                        </tr>
                    </table>

The CSS code looks like this...

                        
table{
    width: 100%;
}

table,th, td{
    border:2px solid black;
    border-collapse: collapse;
    text-align: left;
}

tr, th, td{
    height: 2em;
}

#rowOne{
    background-color: antiquewhite;
}

#rowTwo{
    background-color: #d7fae2;
}

#rowThree{
    background-color: #ded7fa;
}

#rowFour{
    background-color: #fad7df;
}
