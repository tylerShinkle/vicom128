HTML Introduction

HTML stands for Hyper Text Markup language. It is the scaffolding and structural basis of every web page you see. HTML uses tags to descripe different types of elements and specify attributes of those elements. For instance, to describe a paragraph or block of text in HTML you would use the opening <p> tag and closing </p> tag. In between these tagas you could place a string of text and it will follow the rules of the <p> tag and will gain any attributes given to the <p> tag. An example of an attribute would be the lang attribute which descibes the intended language of an element. To assign this attribute to a paragraph tag you would do this...

 <p lang="en">Hello</p>

There are many tags each with many attributes, learn more below!

HTML Projects

Project #1 - Hello World!

First, open your editor / IDE of choice and create a file titled index.html

At the top of the document put the document type declaration. This declaration lets the browser know what type of file it is dealing with. It goes like this...

<!DOCTYPE html>

Next is a set of html tags, they contain all HTML elements. Here's how they look...

<html></html>

They should contain everything except the doctype declaration. You can also include a lang attribute, look at the next code example to see what this should look like.

Next under that, put a set of opening and closing head tags. Everything so far should look like this...

<!DOCTYPE html>
<html lang="en">
 <head>
 </head>
</html>

Head tags contain metadata, or data about data. It contains the title, meta tags for SEO, links to style sheets, character sets and more. Notice how all HTML elements inside the HTML tag will inherit the lang attribute of "en" which stands for english. Now all elements will be specified to be in the language of english.

Inside the head tags we are going to put a set of title tags. The title is displayed on browsers tabs. This tag should read...

<title>
 Hello World!
</title>

Our title is "Hello World!"

Next, under the closing head tag and above the closing html tag put a set of body tags...


<body></body>

Body tags hold all of the windows content, anything in the main window of the browser will be within the body tags.

Next we will add an h1 tag, h1 stands for heading one. There are six headings from h1-h6 with h1 being the biggest and h6 being the smallest. Headings are useful for article or page titles, Any important section you want to signify with text a heading is useful for.


<h1>Hello World!</h1>

That's it, your html page named index.html should look as follows...


<!DOCTYPE html>
<html lang="en">
 <head>
  <title>
   Hello World!
  </title>
 </head>
 <body>
  <h1>
   Hello World!
  </h1>
 </body>
</html>

Now you should be able to double click on your index.html file and your 'Hello World!' web page should appear in your default browser! Congratulations!

HTML References

Global Attributes

Global attributes can be added to any HTML element other than the doctype declaration. They are as follows...

 accesskey

  specifies a shortcut key to focus on an element.

 class

  specifies one or more class names for an element.

 contenteditable

  specifies whether you can edit the content of an element or not.

 dir

  specifies the text direction of an element.

 draggable

  specifies whether an element is draggable or not.

 dropzone

  specifies whether the dragged data is copied, moved or linked, when dropped.

 hidden

  specifies that an element is not relevant.

 id

  specifies a unique id for an element.

 lang

  specifies the language of the element's content.

 spellcheck

  specifies whether the element is to have its spelling and grammar checked or not.

 style

  specifies an elements inline CSS style.

 tabindex

  specifies the tabbing order of an element.

 title

  specifies extra information about an element.

 translate

  specifies whether the content of an element should be translated or not.

Tags and Their Attributes

*note: all tags have access to global attributes other than the doctype declaration.

<!DOCTYPE html>

This is known as the doctype declaration and lets the browser know what kind of file it is dealing with. It should be included at the top of all html pages.

<!DOCTYPE html> Attributes

 There are no attributes usually associated with the doctype declaration.

<html>

The html tag contains all html elements other than the doctype declaration. You can think of the html tags as the container for all other html elements.

<html> Attributes

 xmlns

  Specifies XML namespace if needed.

<head>

The head tag contains many important things, such as meta data, links to partner files, the title to your site, viewport specification and much more. We'll learn more about these other tags later.

<head> Attributes

 HTML5 does not support any attributes for the head tag other than the global attributes.

<body>

The body tags contain all elements intended to appear in the main browser window. Think of it as a container for all of your visible content.

<body> Attributes

 HTML5 does not support any attributes for the body tag other than the global attributes.
