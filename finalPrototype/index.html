Learn to Code!

We offer educational resources for multiple languages. Those languages include...

HTML

HTML stands for Hyper Text Markup Language. It creates the structure of web pages and is the basis of all web pages you see. HTML is the first language you should learn if you want to begin building web sites. HTML describes elements with tags, and displays certain tags in certain ways. Everything in the window you see now is described with some sort of HTML tag, from the title in the tab, to the text your reading now. Although the properties of these elements are altered by CSS, learn more about that next!


CSS

CSS is how you change the appearance of HTML elements, it stands for Cascading Style Sheets. You can think of HTML as the buildings scaffolding and CSS as the paint, decor and floorplan. You can change the position, color and particular behaviors of HTML elements with CSS. We recommend you begin learning CSS soon after get the basics of HTML down.


Java Script

Java Script is a scripting language that allows for user interaction and dynamic content. Many buttons and animations are made using Java Script, as well as most custom user content such as loading your name in a page when you login. If HTML is the scaffolding and CSS is the floorplan then you can think of Java Script as the contractors that can change this in real time, depending on the blueprints or instructions they are given. We recommend learning Java Script after you become comfortable with HTML and CSS. Java Script is extremely popular and is what brings the web to life!


PHP

PHP stands for Hypertext Preprocessor, although the acronym is kind of weird. PHP is a server side scripting language. PHP allows clients (browsers) to send requests to the server by calling PHP files, and the server will send a response page. PHP allows you to interact with other software elements on the server such as databases and analytic tools. Many websites you visit utilize PHP to enable communication between the client and server. Once your comfortable with HTML, CSS and Java Script you should consider learning a sever side language such as PHP or Node.js.


Node.js

Node.js is another server side scripting language that is essentially Java Script running outside of a browser thanks to Google's V8 Java Script engine. With Node.js you can do most anything you can with PHP but with the convenience of being able to develop both your front end and back end in the same language, Java Script. With server side languages you can write to files and databases on the server as well as dynamically structuring web pages and much more. Databases are becoming integral in modern day web development, we highly recommend you learn a server side language even if you plan solely on being a front end developer.
