PHP Introduction

PHP stands for Hypertext Preprocessor (although the acronym does not line up.) PHP is a server side scripting language. With PHP clients can interact with servers and the servers in turn can interact with them in a dynamic, specific sense. For example , with PHP users can upload photos or other files, add data to a data base and much more. PHP files can contain text, HTML, CSS, Java Script and of course PHP code. All PHP files end with a '.php' extension. Many popular sites use PHP such as Facebook, Wikipedia, Baidu, Yahoo! and Flickr. Learn more about PHP below.

PHP Projects

Project #1 - Syntax, Variables and Echo

To start off we will create a very simple PHP page. In order for it to work correctly you will need to do one of the following as you cannot just run PHP in a browser as you do HTML.

Enough about that, lets write some code...

 - Upload or FTP your file to a web server with PHP support.

 - Install a web server on your own computer and then install PHP on the server.

*More on why PHP does not just run in a browser...

 PHP does not run in browsers, it is executed on servers. What happens is a browser asks a server to return an HTML page, if a PHP tag is detected the PHP environment or system parses the PHP tag. Once PHP parses the PHP tags it renders or preprocesses the new HTML page accordingly. Once it is ready it sends the pre processed page to the client. PHP more or less produces HTML, it is not renderable by browsers itself. The PHP code requires the PHP interpreter which runs on servers

First thing is first, PHP code goes in PHP tags like this...

<?php
//code here
?>

Now to declare a variable in PHP you use the dollar sign or '$'. We will create a string called $hello and set it to the message we want to display, like this...

<?php
 $hello = "Hello World";
?>

Now to print this variable to the page you will use the echo keyword. Now our first PHP tag will be done and it will look like this...

<?php
 $hello = "Hello World";
 echo $hello;
?>

**note: In order for a browser to interpret this it needs to be within HTML tags like this...


<!DOCTYPE html>
<html>
<body>
 <?php
  $hello = "Hello World";
  echo $hello;
 ?>
</body>
</html>

You can now host this and see how it works! Then maybe experiment a bit.

PHP References

Important Keywords

 echo

  The echo keyword will output a string of text to the browser.

 static

  The static keyword has to do with declaring and setting properties of functions and variables. Static means that the variable or function is not linked to any object but rather to a class. Static entities are 'more free' or are easier to access. This is useful for dealing with or altering properties of the class itself rather than a specific element of the class.

 public

  The public keyword in PHP has to do with visibility. Properties, methods and constants all have visibility, whether it be public, protected or private. Public is the mos accessible and lenient as public entitities can be accessed from anywhere as the other visibility values have access limitations.

 try

  The try keyword in PHP has to do with exception handling. Exception handling has to do with attempting to execute certain statements, if they operate correctly no exception is thrown and the code executes as normal. If an exception is caught a special block after the try block is executed. The catch block is closely association with the try block and will create an exception object which can be given it's own statements to execute. This helps ensure predictability in the use of code and helps log errors.

 if

  PHP like many other languages has an if statement. The if statement checks a condition and if it is true the code block within the if statement is executed.
