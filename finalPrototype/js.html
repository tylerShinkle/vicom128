JavaScript Introduction

JavaScript is a client side scripting language. It is associated with a HTML page and can manipulate the elements within the HTML page. There are also events in Java Script that allow users to interact with the page. Java Script enables users to interact with the web page and change it. Developers can also create dynamic content with Java Script that changes in different circumstances. There is a lot you can do with Java Script!

Java Script Projects

Project #1 - Variables and the DOM

First, before you can use Java Script you have to link your HTML document to it. Do that by placing the following script tag at the very bottom of the HTML body.

<script src="scripts.js"></script>

*note: replace the value of the src attribute with the path / name of your js file. Java Script files need to have the js extension.

To create a variable in Java Script you use the keyword var followed by whatever name you would like to give the variable provided they don't contain spaces, begin with a letter, underscore or dollar sign and contain no characters other than letters, numbers, underscors or dollar signs. Keep in mind that variables are case sensitive as well. Below we will declare a variable x and set it's value to 5.

var x = 5;

Now we have a variable known as x and can do what we want with it just by referring to it by it's name. Keep in mind you should always use semicolons at the end of a Java Script statement.

Next we will reference the DOM. DOM stands for Document Object Model, which is a programming interface for HTML and XML. With the DOM programs can interact with and change HTML or XML files. For instance, if there were an element with the id #colorMeBlue we could change the color using Java Script like this...

var x = document.getElementById("colorMeBlue");
x.style.color = "blue";

You can do all sorts of stuff with HTML and Java Script, new lessons are coming soon!

Java Script References

Important Keywords

Keywords cannot be used as variable names, function names or anything else. Java Script reserves these words for special purposes.

 var

  var is a keyword used to declare variables.

 new

  new is used by Java Script to create new instances of objects.

 return

  The return keyowrd stops a function and returns a value from that function. If no return value is specified it returns undefined.

 if

  The if keyword in Java Script is a conditional statement. If a certain condition is true then a block of code will be executed. Often paired with else (see below).

 else

  The else keyword follow the code block of an if statement, if the condition is false the code block associated with else will be executed instead.

 for

  The for keyword creates a loop with depending upon three statements. The initialization of a counter, a condition and an expression modifying the counter.

 do

   The do keyword is used to create a loop similar to that of a for loop, except that the code block is guaranteed to execute at least once because the condition is not evaluated until after the the code block is executed. Learn more about these in a future lesson...

 while

  while is used in conjunction with do to form a do while loop. While is followed by a condition at the end of the do code block. The do code block will execute at least once and continue to execute until the while condition is false.
