Relative Position

When you set an elements position to relative it is positioned relative to its normal position. For example if you want an element to shift 20 pixels down you could set it's position to relative and then add a top attribute with the value of 20px. Click here to move this whole artice 20 pixels down and back again.You can think of it like the relative element is getting pushed or pulled by the directions you specify. In the example here the top is pushing the element down by 20 px, you could also do top: -20px; and the element will be pulled up by 20 px. Positive is push and negative is pull.

The code

element{
   position:relative;
   top:20px;
}
