Absolute Position

Absolute postitioning positions the element relative to it's first non-static positioned ancestor.This is useful for positioned containers and arranging elements within them. For instance you could position this article so it is at the top-right of it's parent element. Click here to move this element to the top-right and back. Just like fixed position the margins still apply as they are part of the element.
