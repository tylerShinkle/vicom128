Positioning in CSS

Positioning in CSS allows you to place elements wherever you like. It also controls aspects of how elements behave when a user scrolls or resizes the window.You can manipulate elements by adding a position property and setting it to one of 5 values. (other than initial and inherit) The five position values are static, relative, fixed, absolute, and sticky. Click on the links in the navigation bar above to learn more.
