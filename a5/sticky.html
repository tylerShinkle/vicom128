Sticky Position

Sticky position does not alter the initial position of the element it is applied to. Instead it changes how the element behaves when it reaches a position in the page. For instance the navbar on this website will "stick" to the top of the page once it reaches the top:0; position. As most of these pages are small I've added a div to this page purely to take up space so you can see how this works. Try scrolling and watch the navbar's behavior.

Note: The z-index is set to 1 for the navbar so it is always accesible above all the other elements. Also, this way when content scrolls past / above the navbar, the navbar is not obscured or obstructed.

The code

element{
  position:sticky;
  top:0;
  z-index:1;
}
